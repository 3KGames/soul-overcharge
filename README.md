# Пока гайды не закончены, список используемых плагинов
1. Unitask - более крутые корутины
2. DOTween - очень вкусные анимации через скрипты
3. VContainer - Dependency Injection
4. Fast Script Reload - позволяет изменять скрипты прямо в Play Mode (так же есть экспериментальные функции для более быстрой компиляции, но они вроде почти никогда нормально не работают). Если привыкнуть - имба
5. Naughty Attributes - просто имбовые атрибуты для отображения в инспекторе.
**Каждый имеет свою хорошую документацию**
